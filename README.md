## 운영방식
* 연습문제는 자신이 하고싶을 때 원하는 만큼 하기
* 연습문제 제외 2문제 이상 개별문제 선정(다른 사람들이랑 안 겹치게)
* 공통 1문제 + 개별문제 발표 : PPT, 글, 코드 등 상관 없음
* 강의 15개(이것이 취업을 위한 코딩테스트다 with 파이썬) + 책 참고
* 매주 수 7시 모임
* 프로그래머스 사용
* 다른 사람이 푼 문제는 바로 다음 주에 풀지 말지
* 다른 사람 문제 읽어오기
